# NYC Taxi Data Analysis (NumPy Project)

## Overview
This project performs exploratory data analysis on the NYC Taxi dataset using Python and NumPy. The objective is to extract meaningful insights from real-world trip data such as ride patterns, trip speed, tipping behavior, and airport drop-offs.

## Dataset
- **File:** nyc_taxis.csv  
- Contains NYC taxi trip records including pickup/dropoff locations, trip duration, distance, fare, and tip information.

## Analysis Performed
- Computed average trip speed using distance and time
- Analyzed total rides taken in February
- Identified rides with tips greater than $50
- Filtered rides with drop-offs at JFK airport

## Key Insights
- Average speed helps understand traffic flow patterns in NYC
- February ride distribution gives insight into monthly demand
- High-tip rides are rare but represent premium trips
- JFK airport rides highlight airport traffic activity
